# Credit-Card-Fraud-Detection-with-Machine-Learning
Este projeto tem como objetivo detectar fraudes em transações de cartão de crédito utilizando técnicas de Machine Learning, com foco na comparação entre os modelos SVM e XGBoost.  A base de dados apresenta forte desbalanceamento (~0,17% de fraudes), exigindo o uso de técnicas específicas como SMOTE e métricas adequadas para avaliação.
